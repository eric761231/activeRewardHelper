# 還原 Google Sheet 資料說明

## 功能說明

如果某些行的「已發放」欄位被誤處理（填入了 V），可以使用還原工具將其還原為空白。

## 使用方法

### 方法 1：互動模式（推薦）

```bash
python restore_sheet_rows.py
```

然後按照提示輸入要還原的行號（用逗號分隔）。

**範例**：
```
請輸入行號（用逗號分隔）: 141,230,231,232
```

### 方法 2：命令行模式

```bash
python restore_sheet_rows.py 141 230 231 232
```

直接提供行號作為命令行參數。

## 注意事項

1. **行號範圍**：行號必須是 2 或以上（第 1 行是標題）
2. **已空白**：如果某行已經是空白，會自動跳過
3. **確認操作**：在互動模式下，執行前會要求確認
4. **原值顯示**：還原時會顯示該行的原值

## 安全提示

- 還原操作會直接修改 Google Sheet
- 建議先確認要還原的行號
- 可以分批還原，避免一次處理太多行
